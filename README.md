Versions used:
Node: 8.9.1
Npm 5.5.1
Angular 5.0.2/1.6.1
Angular CLI 1.5.3

Install:
Npm 'npm install -g @angular/cli' -g = globally

Running app local:
on cmd, cd to the folder of checkout (you should see folders e2e, node_modules, src)
Run cmd 'ng serve --open' --open means it opens in browser automatically
